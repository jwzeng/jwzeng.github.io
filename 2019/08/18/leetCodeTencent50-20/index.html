leetcode腾讯精选练习50题（15. 三数之和）

Diposting di 2019-08-18 | Edited on 2019-08-19

题目描述

给定一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组

示例 1:

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/3sum

解题思路

暴力计算，果然超时，想了一会也只想到分成正的和负的两组再暴力结算，时间估计快不了多少。

最后参考了一下别人的：

  • 先将数组进行排序
  • 从左侧开始，选定一个值为 定值
  • 定义首和尾
  • 首尾与 定值 相加
    • 等于 00，记录这三个值，首部右移，尾部左移
    • 小于 00，首部右移
    • 大于 00，尾部左移
  • 定值右移，重复该步骤
